# AutoVault

Premium vehicle database and luxury marketplace. Single HTML file — no build tools, no server, no dependencies.

## Quick Start

1. **Download:** Click the green **Code** button above, then **Download ZIP**
2. **Unzip** the folder
3. **Open** `index.html` in Chrome (double-click it)

That's it. Everything runs locally in your browser.

## Live Demo

**[davidshanfeld.github.io/car-database](https://davidshanfeld.github.io/car-database/)**

## What's Inside

- **30 premium vehicles** (2023-2026) — Porsche, Ferrari, McLaren, BMW M, AMG, Corvette, Audi RS, Tesla Plaid, Lucid Air, and more
- **9 upcoming/concept cars** with expected delivery dates
- **Luxury marketplace** — 9 listings near Newport Beach (under $100k, original MSRP $250k+) with interactive dark-themed map
- **15-question personalization quiz** saved to localStorage
- **Side-by-side comparison** (up to 4 vehicles)
- **Favorites** persisted to localStorage
- **Full spec sheets** — MSRP, market price, HP, packages, fuel economy, warranty, cargo, seating
- **7-dimension scoring** — overall, performance, safety, reliability, comfort, tech, value
- **Live market ticker** refreshing every 4 seconds
- **Cost of ownership** — annual insurance, fuel, maintenance estimates
- **Depreciation projections** — new through 5-year value charts
- **Similar vehicles engine** — tag and price proximity matching
- **Price verification links** — every car links to Car and Driver for independent MSRP confirmation

## Tech

Single self-contained HTML file. Only external dependency is Leaflet.js (CDN) for the marketplace map. All data, styles, and logic are inline.
